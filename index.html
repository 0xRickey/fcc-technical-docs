Conditional Statements

In Java, controlling the flow of your program is vital, and this is achieved using conditional statements like if, else if, and else. The if statement evaluates a condition, executing its block of code when true. If additional conditions need checking, the else if comes into play, only being evaluated if previous conditions are false. The else clause serves as a catch-all, executing its block when no prior conditions hold true. Together, these statements offer a structured and intuitive approach to decision-making in your Java programs.

Examples

If statement:

int age = 25;
if (age >= 18) {
  System.out.println("You are an adult.");
}

Else statement:

int temperature = 28;
if (temperature > 30) {
  System.out.println("It's hot outside.");
} else {
  System.out.println("It's not too hot.");
}

Else-if

int score = 75;
if (score >= 90) {
  System.out.println("You got an A.");
} else if (score >= 80) {
  System.out.println("You got a B.");
} else if (score >= 70) {
  System.out.println("You got a C.");
} else {
  System.out.println("You need to improve your score.");
}
Arrays

In Java, an array is a fundamental data structure that allows you to store a collection of elements of the same data type under a single variable name. Arrays provide a way to efficiently manage and access multiple values of the same type in a contiguous block of memory. Each element in an array is accessed using an index, which starts from 0 for the first element and increments by one for each subsequent element. Arrays are particularly useful when you need to work with a fixed number of elements and want to access them using their positions within the collection.

Declaration

To declare an array in Java, you need to specify the data type of the elements it will hold, followed by the array variable name and square brackets. The general syntax for declaring an array is:

dataType[] arrayName;
// For example to declare an array that holds integers
int[] numbers;

Initialisation

There are a few ways to initialize an array in Java. One common way is to specify the size of the array and assign values to its elements using curly braces and commas:

int[] numbers = {1, 2, 3, 4, 5};

You can also initialize an array using the new keyword along with the size of the array:

int[] numbers = new int[5]; // Creates an array of 5 integers

Array Properties

Arrays in Java have a few important properties:

  • Fixed Size: The size of an array is fixed upon initialization, and it cannot be changed during the program's execution. To accommodate a different number of elements, you would need to create a new array with the desired size.
  • Contiguous Memory: Elements in an array are stored in contiguous memory locations. This characteristic leads to efficient memory usage and allows for constant-time access to elements using their indices.
  • Homogeneous Elements: Arrays can only hold elements of the same data type. If you need to store different data types, you would typically use arrays of objects or other collection
  • Zero-Based Indexing: Array indices start from 0. The index of the last element in an array of size n is n - 1.
  • Length Property: Arrays have a built-in length property that gives you the number of elements in the array. It's particularly useful when iterating over array elements.
While Loops

In Java, a while loop is a fundamental control structure that allows you to repeatedly execute a block of code as long as a specified condition remains true. It provides a way to automate repetitive tasks, iterating over a sequence of actions until the condition evaluates to false. The while loop is especially useful when the number of iterations is not known beforehand and depends on dynamic conditions.

Syntax

while (condition) {
  // Code to be executed repeatedly as long as the condition is true
}

Example

int count = 0;
while (count < 5) {
  System.out.println("Count: " + count);
  count++; // Increment the count for the next iteration
}
For Loops

In Java, a for loop is a versatile control structure designed to efficiently execute a block of code repeatedly for a predetermined number of iterations. For loops are particularly useful when you know the exact number of times you want to repeat an action. They provide a compact and structured way to manage loops while maintaining control over initialization, condition checking, and iteration updates.

Syntax

for (initialization; condition; update) {
  // Code to be executed in each iteration
}

Example

for (int i = 0; i < 5; i++) {
  System.out.println("Iteration: " + i);
}

In this example, the for loop iterates five times. The loop starts with i being 0, and in each iteration, it checks if i is less than 5. If true, it executes the code block and then increments i by 1. The loop stops when i becomes 5.
